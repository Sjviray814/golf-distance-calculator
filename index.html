Sjviray's Golf Distance Calculator
Hello! Welcome to my Machine Learning-powered Golf Distance Calculator! The dataset for this project was taken from my very own Skytrak launch monitor using shots from me and my brother. I trained and implemented an ml5 Neural Network using our shot data to calculate how far a ball should travel given specific stats.


Note:

Since the dataset for this project is relatively small and taken solely from Skytrak, using another launch monitor will likely produce a different result. However, when tested on Skytrak launch monitor data, 80% of predictions are accurate within 5 yards, and over 90% are accurate within 10 yards.
